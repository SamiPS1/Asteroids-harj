# Asteroids

Pieni Pygame-pohjainen asteroids-peli, jossa on pyörivä alus, asteroideja spawnaava kenttä, ampumistoiminto ja yksinkertainen törmäysten hallinta.

## Mitä tämä projekti on

Ohjaat alusta mustassa avaruudessa ja yrität selviytyä, kun asteroidit tulevat jatkuvasti. Ampu niitä hajottaaksesi ja tuhotaksesi ne, ja vältä törmäämistä niihin.

## Kuinka pelin käynnistää

Projektikansiosta:

```bash
./.venv/bin/python main.py
```

## Ohjaimet

- A / D: käänny vasemmalle / oikealle
- W / S: liiku eteenpäin / taaksepäin
- Välilyönti: ammu

## Tekniikka

- Python 3
- Pygame

## Tavoite

Pysy hengissä mahdollisimman pitkään ja räjäytä asteroidikenttä palasiksi.
